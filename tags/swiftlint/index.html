SwiftLint

Realm에서 제공하는 Swift 전용 Lint이다.

Lint란 소스 코드를 분석하여 오류, 스타일 오류, 의심 스러운 구조등을 찾아 표시해주는 도구를 가리킨다.

SwiftLint는 스위프트 스타일 및 컨벤션을 강제하기 위한 도구로 사용되며, Ray Wenderlich 스위프트 스타일 가이드에 대략적인 기반을 두고 있다.



설치 방법

CocoaPods을 이용한 설치 방법을 다루고자 한다.

Podfile에 ‘SwiftLint’를 추가해주고 pod install 명령어를 실행한다.



SwiftLint 적용

  1. 파일 네비게이터에서 app target > Build Phases 선택

  2. +(add a new build phase) > New Run Script Phase 선택

  3. 새로 생성된 Run Script에 아래 스크립트 추가

  4. Cmd + B를 눌러 빌드를 돌리면 Warning과 Error가 쏟아진다.
