Hexo를 이용한 깃블로그 만들기

나만의 기술 블로그를 만들기 위해 여러 블로그 프레임워크를 찾던 중 Hexo프레임워크가 가장 눈에 띄었다.

오늘은 Hexo를 이용한 GitHub blog 만들기에 대하여 포스팅 하고자 한다.


Hexo 설치

hexo 프레임 워크를 전역 설치한 후 원하는 디렉토리로 이동하여 블로그 모듈을 설치 해준다.

모듈 설치 후 hexo server명령어를 통해 로컬 서버를 바로 구동시킬 수 있다.

http://localhost:4000로 접속 해보자.



테마 적용

hexi.io/themes에서 원하는 테마를 다운로드 하고 압축을 풀어 hexo 폴더 내 themes안에 넣어준다.

자세한 설정은 해당 테마 설치 document에 명시되어 있으며, 설정이 완료되면 hexo server명령어로 서버를 재 구동 시켜준다.


패키지 에러가 발생한다면 해당 테마에서 필요로 하는 패키지를 모두 설치해주면 된다.



첫 포스팅 하기

새로운 포스팅을 하기 위해 .md 파일을 생성해 준다.

마크다운 문법은 MarkDown 사용법 총정리 페이지를 참고!



깃허브 저장소 생성

헥소 블로그는 두개의 저장소를 사용하게 되는데,

하나는 사용자가 보게될 배포 페이지용 다른 하나는 실제 소스를 저장할 저장소 이다.

쉽게 말해 프론트/백엔드로 구분된다.


깃허브로 이동하여 저장소 이름을 사용자계정.github.io 로 생성 해준다.

생성 할 때 README.md파일도 같이 생성 해준다.


저장소를 생성하고 https://사용자계정.github.io로 접속하면 페이지가 정상적으로 출력되는것을 확인할 수 있다.



Hexo-Git 연동

hexo폴더 내 _config.yml파일을 수정해준다.

깃허브 페이지로 배포하기 위해 배포 플러그인을 설치해 준다.
