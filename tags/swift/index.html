[Swift] UIBarButtonItem을 이용한 ToolBar 구현

UIBarButtonItem을 사용하여 아이콘 버튼으로 이루어진 Bottom Tool Bar를 구현 하고자 한다.

아이콘 버튼 생성

버튼을 생성하고 이미지와 touchUpInside Event를 추가 해주었다.

버튼에 Tag를 적용 시켜 어떤 버튼이 눌렸는지 구분해주면 좋다.

버튼에 UIView를 사용하여, Badge를 추가 하거나, Title을 설정해 텍스트도 노출 시킬 수 있으니 활용 해보자.

터치 이벤트 구현

UIBarButtonItem 생성

UIBarButtonItem을 생성 할 때 CustomView를 사용하여 생성해 준다.

미리 생성해둔 아이콘 버튼을 사용한다.
